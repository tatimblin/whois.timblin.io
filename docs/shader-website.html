01 — TECHNIQUE

SVG feTurbulence

The same noise function behind GLSL shaders, exposed as a CSS filter. Applied to <body> it warps every element.

02 — SYNC

Shared Clock

JS updates the SVG seed value each frame. Your Three.js uniform and the CSS filter use the same time value.

03 — CONTROL

Per-element

Override the body filter on any element. Cards can wobble differently from text, borders from fills.
